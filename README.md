# InterX Backend

This is the backend server for InterX application built with Node.js and Express.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the server:
   - For development:
     ```bash
     npm run dev
     ```
   - For production:
     ```bash
     npm start
     ```

The server will run on port 5000 by default. You can change this by modifying the `.env` file.
